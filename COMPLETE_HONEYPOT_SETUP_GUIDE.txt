## TROUBLESHOOTING GUIDE

### Issue: "Port 8501 already in use"
# Solution: Kill existing process
netstat -ano | findstr :8501
taskkill /PID [PID_NUMBER] /F
# Then restart: .\scripts\run_demo.bat

### Issue: "Python packages not found"
# Solution: Activate virtual environment first
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

### Issue: "Vagrant VMs won't start"
# Solution: Check VirtualBox and Vagrant
vagrant --version
VBoxManage list vms
# Try: vagrant plugin install vagrant-vbguest

### Issue: "Docker containers fail"
# Solution: Clean up and restart
docker system prune -a
docker pull ubuntu:focal

### Issue: "GeoIP database missing"
# Solution: Download from MaxMind website
# Place at: C:\project\data\GeoLite2-Country.mmdb

### Issue: "Git clone fails"
# Solution: Check repository URL and permissions
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

### Issue: "Streamlit won't start"
# Solution: Check if port is free and venv is activated
netstat -ano | findstr :8501
.\.venv\Scripts\Activate.ps1
python -m streamlit run app_auto.py --server.port 8501
